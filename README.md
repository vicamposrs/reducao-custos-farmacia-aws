# Redução de Custos em Farmácias com AWS

Projeto desenvolvido como parte do bootcamp **TOTVS - Fundamentos de Engenharia de Dados e Machine Learning**, da DIO.

## Objetivo

Propor uma solução baseada em serviços AWS para reduzir custos operacionais de uma empresa farmacêutica fictícia que atua como hub de distribuição.

## Serviços AWS utilizados

- Amazon S3
- Amazon RDS
- Amazon EC2
